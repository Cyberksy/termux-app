# Dominion University Smart Portal (DUSP)

## Stack
- PHP 8+
- MySQL 8+
- HTML/CSS/JavaScript

## Setup (XAMPP)
1. Copy `dusp` folder to `htdocs`.
2. Import `database/dusp.sql` in phpMyAdmin.
3. Update DB credentials in `includes/config.php`.
4. Visit `http://localhost/dusp`.
